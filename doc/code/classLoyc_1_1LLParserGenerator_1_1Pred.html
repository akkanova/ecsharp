Remarks

Represents part of a grammar for the LLParserGenerator.

This class is the root of a class hierarchy which contains

  • TerminalPred: represents a terminal (which is a token or a character) or a set of possible terminals (e.g. 'A'..'Z').
  • RuleRef: represents a nonterminal, which is a reference to a rule.
  • Other components of a rule: sequences (Seq), branches and loops (Alts), gates (Gate, a mechanism to separate prediction from matching), and and-predicates (AndPred, an assertion that consumes no input).
  • EndOfRule: a container for the follow set of a Rule (this class is not a real predicate; it is derived from Pred so that it can be a legal value for Pred.Next).

Each Pred object can be used only once in a grammar, because Preds contain context- sensitive state such as the Next field, which are used during grammar analysis. A Pred must be Clone()d if one wants to use it multiple times.
